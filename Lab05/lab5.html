Descripción

En esta actividad se hará una introducción a JavaScript.

Objetivos:

  • Conocer el propósito y las características principales de JavaScript
  • Conocer las reglas básicas de sintaxis de JavaScript
  • Elaborar programas básicos con JavaScript

Instrucciones

  1. Revisa la presentación Fundamentos de JavaScript
  2. Escribe, prueba y debuguea (si es necesario) scripts de JavaScript para los siguientes problemas. Cuando se requiera escribir funciones, es necesario incluir un script para probar la función con al menos 2 conjuntos de datos de prueba. Haz las pruebas en un documento.
    1. Entrada: un número pedido con un prompt.
      Salida: Una tabla con los números del 1 al número dado con sus cuadrados y cubos. Utiliza document.write para producir la salida
    2. Entrada: Usando un prompt se pide el resultado de la suma de 2 números generados de manera aleatoria.
      Salida: La página debe indicar si el resultado fue correcto o incorrecto, y el tiempo que tardó el usuario en escribir la respuesta.
    3. Función: contador.
      Parámetros: Un arreglo de números.
      Regresa: La cantidad de números negativos en el arreglo, la cantidad de 0's, y la cantidad de valores mayores a 0 en el arreglo.
    4. Función: promedios.
      Parámetros: Un arreglo de arreglos de números.
      Regresa: Un arreglo con los promedios de cada uno de los renglones de la matriz.
    5. Función: inverso.
      Parámetros: Un número.
      Regresa: El número con sus dígitos en orden inverso.
    6. Crea una solución para un problema de tu elección (puede ser algo relacionado con tus intereses, alguna problemática que hayas identificado en algún ámbito, un problema de programación que hayas resuelto en otro lenguaje, un problema de la ACM, entre otros). El problema debe estar descrito en un documento HTML, y la solución implementada en JavaScript, utilizando al menos la creación de un objeto, el objeto además de su constructor deben tener al menos 2 métodos. Muestra los resultados en el documento HTML.

Preguntas a responder

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Diferencias

      • JavaScript tradicionalmente ha sido un lenguaje interpretado, y Java es compilado.
        La principal diferencia entre un lenguaje compilado y uno interpretado es que el lenguaje compilado requiere un paso adicional antes de ser ejecutado, la compilación, que convierte el código que escribes a lenguaje de máquina. Un lenguaje interpretado, por otro lado, es convertido a lenguaje de máquina a medida que es ejecutado.[2]
      • Java se depura en dos fases y JavaScript en una.
        Java tiene dos fases donde se puede detectar errores cuando se compila y cuando se ejecuta; Mientras que JavaScript es al mismo tiempo.[1]
      • Java es un lenguaje orientado a objetos puro, pero JavaScript está basado en prototipos.[1]
      • Java es fuertemente tipado, y JavaScript es débilmente tipado.[1]
        Tipado es cuando no se permiten violaciones de los tipos de datos, es decir, dado el valor de una variable de un tipo concreto, no se puede usar como si fuera de otro tipo distinto a menos que se haga una conversión.[3]
      • JavaScript es estándar, Java no.[1]

    • Semejanzas

        • Ambos soportan orientado a objetos.

  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    1. getTime() Obtiene el tiempo millisegundos desde Enero 1, 1970
    2. getDate() Regresa el dia.
    3. getFullYear() Regresa el año
    4. getUTCDay() Regresa el dia de la semana, deacuerdo al timpo universal( 0-6)
    5. now() Regresa el tiempo en mili segundos des enero 1, 1970

  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    1. toString() Convierte una array en string y regresa el resultado
    2. concat() Junta dos arrays y regresa el resultado
    3. find() Regresa el primer elemto de un array que pase una prueba
    4. indexOf() Busca un elemento de la array y regresa su posicion
    5. isArray() Revisa si un elemento esta en la array
