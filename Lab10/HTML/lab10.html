Laboratorio 10

RUTAS Y FORMAS

Descripción

En esta actividad haremos una introducción al desarrollo en el back-end con node.En esta actividad veremos cómo programar desde el servidor un módulo simple de ruteo y cómo mandarle datos al servidor.

Objetivos:

  • Entender cómo responder desde el servidor a las diferentes rutas para poder diseñar y programar el ruteo de una aplicación.
  • Preparar el ambiente de trabajo para hacer desarrollo en el back-end.Entender las diferentes maneras de enviar datos a un servidor por medio de HTTP, para poder realizar un procesamiento con estos datos.
  • Entender cómo funciona el ambiente de ejecución de Node.js, y su forma asíncrona de manejo de eventos.

Instrucciones

  1. Sigue la demostración del profesor en la sesión de clase.

  2. Utilizando alguno de tus laboratorios anteriores (o si lo prefieres empieza de 0), elabora una aplicación web con las siguientes características:
    1. La aplicación debe poder responder al menos a 3 rutas diferentes, y mandar una respuesta HTTP 404 cuando la ruta no existe.
    2. En alguna de las rutas, la aplicación debe contener al menos 1 forma que debe enviar datos al servidor por POST. El servidor debe reaccionar ante este envío y guardar los datos en un archivo de texto dentro del mismo servidor.
