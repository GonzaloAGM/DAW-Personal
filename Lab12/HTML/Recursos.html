Preguntas a responder

  • Describe el archivo package.json.

    • Este es un archivo que administra las dependencias de nuestro proyecto web. Este indica cosas como que paquetes son necesarios instalar en la carpte node modules para que el proyecto sea funcional (esto realizandose automáticamente cuando se realiza el npm install).
    Al mismo tiempo, funciona como "manifiesto" del proyecto general, es decir, se define nombre, versión, repositorio, autor, licencia, etc.
