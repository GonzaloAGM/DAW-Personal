Laboratorio 7

FRONT-END FRAMEWORKS

Descripción

En esta actividad exploraremos algunos de los frameworks para desarrollo de Front-end de aplicaciones web, y aprenderás a usar uno de estos frameworks para mejorar la experiencia del usuario.

Objetivos:

  • Que conozcas frameworks para el desarrollo de front-end, y aprendas a usar uno de ellos.
  • Desarrollar tu capacidad de autoaprendizaje.
  • Que amplíes tu abanico de librerías para generar interfaces web amigables para el usuario.

Instrucciones

  1. Revisa los siguientes enlaces a sitios de frameworks de Front-end:
    • Bootstrap
    • Foundation
    • Materialize
    • Bulma

  2. Selecciona uno de los frameworks.

  3. Aplica el framework y sus componentes a alguno de tus laboratorios o proyectos anteriores, al avance que lleves de tu proyecto actual, o bien elabora una pequeña aplicación.

  4. Responde a las preguntas dentro de tu aplicación.
